# Sistema de Gestión de Veterinaria - "Ralah Pets"

Este es un sistema desarrollado para la gestión de una clínica veterinaria, permitiendo el control de citas, historial médico de mascotas, clientes y productos.

## Tecnologías utilizadas
* **Backend / Frontend:** PHP, HTML5, CSS3, JavaScript (coloca aquí los que usaste).
* **Servidor Local:** XAMPP (Apache).
* **Base de Datos:** MySQL (phpMyAdmin).

## Instrucciones de Instalación y Ejecución (Local)

Para correr este proyecto en tu computadora local, sigue estos pasos:

### 1. Requisitos Previos
* Tener instalado [XAMPP](https://www.apachefriends.org/).

### 2. Configuración del Código
1. Descarga este repositorio como un archivo ZIP (o clónalo).
2. Extrae el contenido y coloca la carpeta del proyecto dentro del directorio de XAMPP:
   `C:\xampp\htdocs\VETERINARIARALAHPETS`

### 3. Configuración de la Base de Datos
1. Abre el **XAMPP Control Panel** e inicia los servicios de **Apache** y **MySQL**.
2. Ve a tu navegador e ingresa a [http://localhost/phpmyadmin](http://localhost/phpmyadmin).
3. Crea una nueva base de datos llamada `veterinaria_x`.
4. Selecciona la base de datos creada, ve a la pestaña **Importar**, selecciona el archivo ubicado en la carpeta del proyecto: `/database/veterinaria_x` y haz clic en **Importar**.

### 4. Ejecución
* Abre tu navegador e ingresa a la siguiente dirección:
  `http://localhost/VETERINARIARALAHPETS`

### 5. Usos
* Existen 2 tipos de usuarios, los **Usuarios normales**, clientes que se registran en la página, login y register normal y tenemos al usuario **Administrador**
* User y contraseña del **Administrador**,  User: admin@ralahpets.com  Password: Admin123!
